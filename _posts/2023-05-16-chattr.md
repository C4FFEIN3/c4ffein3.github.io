## 용도
- 디렉토리 혹은 파일의 속성 변경

## 사용방법
### 설정

|옵션|내용|
|---|---|
|+|특정 속성 부여
|-|특정 속성 제거|

### 속성
|속성|내용|
|---|---|
|a|append만 가능|
|c|압축된 상태로 저장|
|d|dump로 백업 불가|
|i|read-only 상태|
|s|파일 삭제시 sync|
|S|파일 변경시 sync|
|u|삭제된 경우 그 내용이 저장되어 삭제되기 전 데이터로 복구 가능|

### 예시
```bash
$ chattr +i a.txt
```
- a.txt 수정 불가

```bash
$ chattr -i a.txt
```
- a.txt 수정 가능
